게임 이름: 야생 대결 (Wild Duel)

[게임 개요]
- 모든 플레이어는 게임 시작 시 지정된 월드 스폰에서 시작합니다.
- 시작 아이템으로 돌 도끼와 돌 곡괭이가 지급됩니다.
- 게임은 총 2단계로 구성됩니다:
  1. 파밍(준비) 시간 (15분): 이 시간에는 PvP가 비활성화되고, 사망 시에도 아이템을 잃지 않고 부활할 수 있습니다. (keepInventory = true)
  2. 전투 시간: 파밍 시간이 종료되면 PvP가 활성화됩니다. 이때부터는 사망하면 관전 모드로 전환되며, 아이템을 모두 잃게 됩니다. (keepInventory = false) 또한, 월드보더가 점차 축소되기 시작합니다.

[명령어 설명 (OP 전용)]

1. /wd set
- 게임을 시작하기 전 대기 상태로 전환하고, 모든 플레이어를 소환합니다.
- 명령어를 실행한 OP의 위치를 게임 대기 장소로 설정합니다.
- 서버에 접속 중인 모든 플레이어를 해당 대기 장소로 텔레포트시킵니다.
- 모든 플레이어에게 '포화' 효과를 부여하여 움직임을 제한합니다.

2. /wd sp
- 명령어를 사용한 OP의 현재 위치를 게임 시작 지점(월드 스폰)으로 등록합니다.
- 게임이 시작되면 모든 플레이어는 이 위치로 텔레포트됩니다.

3. /wd start
- 게임을 본격적으로 시작합니다.
- 모든 플레이어를 등록된 시작 지점으로 텔레포트합니다.
- 기본 아이템(돌 도끼, 돌 곡괭이)을 지급합니다.
- 파밍 단계 설정을 적용합니다 (keepInventory = true, PvP 비활성화).
- 월드보더의 크기를 반지름 1000으로 설정합니다.
- 15분의 파밍(준비) 시간을 시작하며, 남은 시간은 보스바에 표시됩니다.
- 파밍 시간이 종료되면 자동으로 전투 단계로 전환됩니다.

4. /wd ts <초>
- 파밍(준비) 시간이 진행 중일 때, 남은 시간을 지정된 시간(초)으로 강제 설정합니다.

[게임 진행 요약]

1. /wd set (OP): 대기 장소를 설정하고 모든 플레이어를 소환하여 준비시킵니다.
2. /wd sp (OP): 게임이 시작될 스폰 위치를 등록합니다.
3. /wd start (OP): 15분의 파밍 시간을 시작합니다.
4. /wd ts <초> (OP): (선택 사항) 파밍 시간을 조절합니다.
5. 파밍 시간 종료: 자동으로 전투가 시작되고 최후의 1인이 남을 때까지 진행됩니다.

[기타 시스템]

- 파밍 시간 중 사망: `/wd sp`로 지정된 장소에서 아이템을 보존한 채로 자동 부활합니다.
- 전투 시간 중 사망: 관전 모드로 전환되며, 아이템을 모두 잃습니다.
- 월드보더: 전투가 시작되면 월드보더가 반지름 50까지 서서히 축소됩니다.
